## 🎮 Sobre o projeto

Este projeto simula o famoso jogo **Pedra, Papel e Tesoura (Jokenpô)** no terminal.

O jogador enfrenta o computador, que faz escolhas aleatórias, e o sistema define o vencedor com base nas regras tradicionais do jogo.

---

## 🧠 Demonstração de habilidades

✔ Estruturas condicionais (`if / else`)  
✔ Laços de repetição (`while`)  
✔ Tratamento de erros (`try / except`)  
✔ Funções organizadas e reutilizáveis  
✔ Uso de bibliotecas padrão (`random`, `time`)  
✔ Interação com usuário via terminal  

---

## 🕹️ Como jogar

No terminal, escolha uma das opções:


0 → Pedra ✊
1 → Papel ✋
2 → Tesoura ✌️


O computador fará sua jogada automaticamente e o resultado será exibido na tela.
